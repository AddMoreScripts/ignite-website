Machine Learning With TensorFlow

TensorFlow is an open source software library for high-performance numerical computation that is used mostly for deep learning and other computationally intensive machine learning tasks. Its flexible architecture allows easy deployment of computation across a variety of platforms (CPUs, GPUs, TPUs). Originally developed by researchers and engineers from the Google Brain team within Google’s AI organization, it comes with strong support for machine learning and deep learning and the flexible numerical computation core is used across many other scientific domains.

At the same time, Apache Ignite is a memory-centric distributed database, caching, and processing platform for transactional, analytical, and streaming workloads delivering in-memory speeds at petabyte scale. Originally developed by GridGain company it’s now an open source Apache top project used by many companies from “Fortune 500” list. Apache Ignite provides fast access to data and allows to perform distributed computations thus it can be used with TensorFlow in a wide range of use cases.

TensorFlow and Apache Ignite together to provide a full toolset needed to work with operational and historical data, to perform data analysis and to build complex mathematical models based on neural networks.
