Start Contributing

Before you start working on Apache Ignite, you should sign up to the dev mailing list and say "Hello" to everyone in the community. We have a very friendly group and intdend to make joining the project as easy as possible.

You should also sign up to Ignite Jira and browse through open tickets. There are many tasks you can get involved in. Where the most help is needed right now is our streaming and data strucutures functionality.

To Get Started:

  • Choose any of the tickets below.
  • Send an email to the dev list letting us know that you want to work on it (make sure to subsribe first).
  • One of the committers will add you to the contributor list and assign the ticket to you.

Ignite Development

Apache Ignite uses GIT for its VCS system. The "master" branch contains the last released version. The "sprint-N" branch contains the latest code for the Nth sprint.

GIT checkout:
                # Master branch.
                $ git clone https://git-wip-us.apache.org/repos/asf/incubator-ignite

                # Sprint branch.
                $ git clone https://git-wip-us.apache.org/repos/asf/incubator-ignite -b sprint-2
Build Project:
                # Build project.
                $ mvn clean package -DskipTests -P-release

                # Build binary release
                $ mvn clean package -DskipTests

                # Run Apache RAT
                $ mvn clean validate -Pcheck-licenses
To start contributing:
  • Pick a ticket from Ignite Jira
  • Create a branch in GIT (same name as the ticket, all lowercase)
  • When ready, submit a patch in Jira and assign to a committer
  • Send email to the dev list stating that patch is ready
  • The committer will provide comments after review.
  • If the patch is rejected, address comments and repeat the process.
 

Website Development

Apache Ignite uses Bootstrap framework for its website.

Installation:
  • Install SASS (http://sass-lang.com/install) using: 
                                    $ sudo gem install sass
  • Install compass (http://compass-style.org/install) using: 
                                    $ gem install compass
  • Checkout SVN website repository: 
                                    $ svn co https://svn.apache.org/repos/asf/incubator/ignite/site/trunk
To edit CSS:
  • Do NOT edit all.css directly.
  • Run SCSS watchdog: 
                                    $ compass watch scss/all.scss
  • You can edit "all.scss" and "ignite.scss" files.
