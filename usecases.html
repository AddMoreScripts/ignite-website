Apache Ignite Use Cases

As a platform, Apache Ignite is used for a variety of use cases some of which are listed below:

Database

  • Distributed Database
  • In-Memory Database
  • In-Memory Data Grid
  • Key-Value Store

Comparison

  • Ignite for NoSQL Users
  • Ignite for RDBMS Users

In-Memory Caching

  • Database Caching
  • JCache Provider
  • Hibernate L2 Cache
  • Web Session Clustering

Spark & Hadoop

  • Spark Shared RDDs
  • Faster Sql for Spark
  • IGFS as HDFS Cache
  • Faster MapReduce
