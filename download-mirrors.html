Legal

Binary downloads are provided as a convenience for users who cannot or do not wish to compile the source themselves. These binaries are not official Apache Ignite (incubating) releases. We recommended that you download the officially released source distributions and verify the signatures of the downloaded files before opening them.

For more information about Apache release policy see What is a Release?

Building the Binaries

From Git:
       # Check Out Sprint 2 branch.
       $ git clone https://git-wip-us.apache.org/repos/asf/incubator-ignite -b sprint-2

       # Build maven project
       $ cd incubator-ignite
       $ mvn clean package -DskipTests -P-release
From a source package:
       # Unpack the source package
       $ unzip -q incubator-ignite-[srcpackageversion].zip

       # Build the binary release
       $ cd incubator-ignite-[srcpackageversion]
       $ mvn clean package -DskipTests
