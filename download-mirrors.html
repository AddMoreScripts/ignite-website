Binary Downloads

The Apache Ignite (incubating) project provides binaries as a convenience for those who do not wish to compile the source themselves. These binaries are not official Apache Ignite (incubating) releases. We recommended that you download the officially released source distributions and verify the signatures of the downloaded files before opening them.

The binary download includes:

  • startup scripts
  • default configuraiton
  • examples for all major components
  • libraries (required and optional)

Building the Binaries

From Git:
       # Check Out Sprint 2 branch.
       $ git clone https://git-wip-us.apache.org/repos/asf/incubator-ignite -b sprint-2
       # Build the binary release
       $ cd incubator-ignite
       $ mvn clean package -DskipTests
From a source package:
       # Unpack the source package
       $ unzip -q incubator-ignite-[version].zip
       # Build maven project
       $ cd incubator-ignite-[version]
       $ mvn clean package -DskipTests -P-release
