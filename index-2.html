APACHE IGNITE™

a memory-centric multi-model distributed database, caching, and processing platform
for transactional, analytical, and streaming workloads, delivering in-memory speeds at petabyte scale

Ignite is Used by

ING, HomeAway, FSB,
Sberbank, 24 Hour Fitness,
Wellington, JacTravel, The Glue,
and many more

Keep Your Database

Accelerate existing Relational and NoSQL databases with Ignite™ in-memory data grid and caching capabilities

SQL at NoSQL Scale

Achieve horizontal scalability, strong consistency, and high-availability while accessing clustered data using Ignite distributed SQL

 

MAIN FEATURES

Distributed SQL

SQL database with support for distributed joins

Key-Value

Read, write, and transact with fastest key-value APIs

ACID Transactions

Enforce full ACID compliance across distributed data sets

 

Durable Memory

Store and process distributed data
in memory and on disk

Collocated Processing

Avoid data noise by sending computations to cluster nodes

Machine Learning βeta

Train and deploy distributed machine learning models
