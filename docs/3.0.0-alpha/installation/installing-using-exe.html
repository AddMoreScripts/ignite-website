    Installing Using Distributive

    Prerequisites

    Ignite was tested on:

    JDK

    11 and later

    OS

    Linux (Ubuntu 20.04), Windows (10 Home), macOS

    Installing Using Distributive

    To install Apache Ignite, perform the following steps:

    1. Navigate to Apache Ignite Downloads page to download the distributive.

    2. Put the distributed file into the installation folder in your system.

    3. Either execute the file or run it in the command line.

    4. Optionally, set the Windows PATH to point to the installation folder and make sure there is no trailing / (or \ for Windows) in the path.

    Note
    		 Depending on your system, you may need additional permissions during the installation. To grant permissions for Linux, run the chmod +x ignite standard command.
