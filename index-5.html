Ignite is a memory-centric platform

  • combining a distributed SQL database

  • with a key-value data grid

  • that is ACID-compliant

  • and horizontally scalable

Learn more

Star Fork Watch

IN-MEMORY PERFORMANCE WITH DURABILITY OF DISK


Durable Memory

Store and process distributed data in RAM and disk using the new memory-centric architecture.


Distributed SQL

Benefit from distributed SQL database capabilities with ANSI-99 queries, DML, DDL and indexing.


ACID Transactions

Use fully ACID compliant distributed transactions ensuring strong data consistency at all times.

 


Key-Value

Read and write partitioned data sliced over the cluster nearly instantaneously.


Collocated Processing

Avoid data movement across the network by sending computations to cluster nodes.


Machine Learning

Train and deploy machine learning models in cluster to resolve some of the most sophisticated tasks.
